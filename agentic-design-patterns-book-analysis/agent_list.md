# 智能体类型与用例清单 (Agent List)

本文档总结了《智能体设计模式》中提到的各种实际应用和用例中的智能体（Agent）类型及其核心职责。

| Agent 类型 | 核心职责 | 典型应用案例 | 来源章节 |
| :--- | :--- | :--- | :--- |
| **市场分析师 (Market Analyst)** | 总结市场报告、识别趋势并提取关键数据 | 市场调研报告处理工作流 | 第 1 章 |
| **校对员/评审员 (Proofreader/Reviewer)** | 检查内容质量、语气和逻辑，提供改进建议 | 代码审核、高质量内容创作、任务目标校验 | 第 1, 4, 7, 11, 19 章 |
| **路由智能体 (Router Agent)** | 根据意图、语义或资源约束将请求分发给专业子 Agent 或特定模型 | 智能客服意图识别、资源感知型模型切换 (Flash vs Pro) | 第 2, 16 章 |
| **状态评估器 (State Evaluator)** | 监控系统累积状态并指导流程流向 | 复杂动态工作流决策 | 第 2 章 |
| **总调度器 (Main Dispatcher)** | 在拥有多个专业工具的系统中选择最合适的工具 | 动态工具选择场景 | 第 2 章 |
| **并发控制器 (Concurrency Controller)** | 协调多个独立子任务的启动、执行和结果收集 | 多源信息搜集、多维度并行分析 | 第 3 章 |
| **生产者/生成器 (Producer/Generator)** | 执行初始任务（如编写代码、撰写文章、制定计划） | 生产者-评审者模型、代码生成 | 第 4, 11 章 |
| **深度研究助手 (Deep Research Assistant)** | 自主生成研究计划、执行搜索、分析网页并动态修正策略 | 深度行业调研、学术背景搜索 | 第 6, 17, 21 章 |
| **管理者/中心调度员 (Supervisor)** | 决定任务委派顺序，管理专家 Agent 团队 | 复杂项目编排、多 Agent 协作系统 | 第 7, 9, 15 章 |
| **专业技术文案 (Technical Copywriter)** | 负责专业领域的文档撰写与润色 | 软件开发团队、产品说明书编写 | 第 7 章 |
| **脚手架 Agent (Scaffolding Agent)** | 自动生成项目基础结构和代码框架 | 数字化开发团队 | 第 7 章 |
| **测试 Agent (Test Agent)** | 编写并运行单元测试，验证代码正确性 | 自动化编程与调试工作流 | 第 7, 11 章 |
| **记忆服务 Agent (Memory Service)** | 捕获、存储、检索并管理长期记忆与用户偏好 | 个性化客服、多轮任务追踪 | 第 8 章 |
| **自我改进编码智能体 (SICA)** | 能够修改自身代码并进行隔离运行验证 | 软件自我进化、代码优化 | 第 9 章 |
| **异步监督者 (Asynchronous Supervisor)** | 监控智能体的自我改进过程，防止偏离目标或产生安全风险 | 智能体学习与进化过程监控 | 第 9, 21 章 |
| **判定者/裁判 (Judge)** | 对输出结果进行二值判定（True/False）或多维评分 | 任务停止条件判定、LLM-as-a-Judge 评估 | 第 11, 19 章 |
| **主/备处理器 (Primary/Fallback Handler)** | 负责执行核心任务及在失败时的降级方案处理 | 生产级系统异常处理与恢复机制 | 第 12 章 |
| **诊断智能体 (Diagnosis Agent)** | 分析系统失败原因并生成修正提示词 | 自动异常恢复、系统自愈 | 第 12 章 |
| **人类监督员 (Human Overseer)** | 监控 Agent 行为，在关键/高风险环节介入决策 | 医疗/金融 Agent、内容审核 | 第 13 章 |
| **推理 Agent (Reasoning Agent)** | 评估检索质量、解决信息冲突并执行多步逻辑推演 | Agentic RAG、复杂决策支持 | 第 14, 17 章 |
| **资源感知路由器 (Resource-Aware Router)** | 预判任务复杂度并动态分配计算资源 | 追求性价比的商业化 Agent 应用 | 第 16 章 |
| **审计/策略执行角色 (Audit/Policy Agent)** | 强制执行安全边界、合规性检查和权限约束 | 防护栏系统、反偏见审查 | 第 18 章 |
| **优先级排序智能体 (Prioritization Agent)** | 根据紧急度、重要性和资源状况动态调整任务队列 | 应急响应系统、项目管理调度 | 第 20 章 |
| **假设生成器 (Hypothesis Generator)** | 在未知解空间中生成科学假设或创新思路 | 自主科学研究、新药研发 | 第 21 章 |
| **进化引擎 (Evolution Engine)** | 通过变异和组合算法不断演化和优化解决方案 | 方案搜索、算法自动进化 | 第 9, 21 章 |
| **元审查员 (Meta-Reviewer)** | 从高维度评估研究路径的创新性、可行性和伦理合规性 | 科研项目管理、前瞻性探索 | 第 21 章 |

---
*注：本列表基于各章节分析报告整理，涵盖了从基础执行到高级科研探索的全方位智能体角色定义。*
