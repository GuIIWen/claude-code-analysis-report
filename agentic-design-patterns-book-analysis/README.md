# Agentic Design Patterns Book Analysis

- `analysis.md`：全书逐章深度提炼。
- `index.html`：对应的单页展示版，支持筛选与逐章展开。
- `mindmap.md`：中文交互脑图的结构化内容源。
- `mindmap.html`：基于 `mindmap.md` 生成的自包含中文交互脑图，支持搜索、折叠和章节聚焦。
- `mindmap_review.md`：独立 P8 对脑图交付物的检视意见与验收清单。

为了记忆和复盘，建议优先打开 `mindmap.html`；需要脑图内容源时阅读 `mindmap.md`，需要逐章长文版时阅读 `analysis.md`，需要普通展示版时打开 `index.html`。
