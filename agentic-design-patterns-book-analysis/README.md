# Agentic Design Patterns Book Analysis

- `analysis.md`：全书逐章深度提炼。
- `index.html`：对应的单页展示版，支持筛选与逐章展开。
- `analyze.md`：SVG 脑图的结构化内容源，组织为“中心主题 → 4 个 Part → 21 章 → 核心组件 / 关键技术节点 / 记忆锚点”。
- `mindmap.html`：基于 `analyze.md` 生成的自包含 SVG 真脑图；中心节点向 4 个 Part、章节、组件与技术节点发散，支持搜索和章节聚焦。
- `mindmap_svg_review.md`：独立 P8 对 SVG 真脑图交付物的检视意见与验收清单。

为了记忆和复盘，建议优先打开 `mindmap.html`；需要脑图内容源时阅读 `analyze.md`，需要逐章长文版时阅读 `analysis.md`，需要普通展示版时打开 `index.html`。
